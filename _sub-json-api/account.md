# service-login
+ **uri:**  */gr_login*
+ **method:** GET
+ **parameters**

>
|name|required|desc|
|---|---|-|
|jver|YES|api version|
|gr_nm|YES|app group name|
|gr_key|YES|service key|

+ **example**

> http://api.stick.tv/gr_login?jver=2&gr_nm=singlab&gr_key=f7d2d2599d49dd6c755a62314185d98b
